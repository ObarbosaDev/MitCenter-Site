# MitCenter-Site

## Descrição do Projeto

**MitCenter-Site** é um **site institucional estático** desenvolvido como projeto web em **HTML, CSS e recursos visuais** para apresentar informações de uma oficina automotiva chamada *MitCenter*.
O objetivo principal é proporcionar uma **presença online atrativa e informativa**, destacando serviços, imagens e a identidade visual da oficina. Esta versão é fruto de **estudos pessoais e acadêmicos**, com foco em prática de desenvolvimento frontend.

## Demonstração

O site contém as seguintes páginas/elementos:

* **Página principal (`index.html`)** com apresentação da oficina e serviços;
* **Elementos de imagem** como fotos da oficina e trabalhos realizados;
* **Logo e favicon** para reforçar a identidade visual;
* Recursos estáticos (não há backend ou banco de dados).

## Como Visualizar o Site Localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/ObarbosaDev/MitCenter-Site.git
   ```

2. Navegue até a pasta do projeto:

   ```bash
   cd MitCenter-Site
   ```

3. Abra o arquivo `index.html` em um navegador de sua preferência.
